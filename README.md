## Getting started

To get started:

* Install and start the API server
  
  From the root of this repository:
    - `cd api-server`
    - `npm install`
    - `node server`


* In another terminal window, install npm dependencies and start the front end

  From the root of this repository:
    - `cd frontend`
    - `npm install`
    - `npm start`

go to http://localhost:3000 to see the application.
